# Event Management Web App

## Project Overview

This project is an Interactive Event Management Web Application developed using HTML, CSS, and JavaScript.

The application allows users to:

* View upcoming events
* Add new events
* Delete events
* Search events by name or date
* Automatically sort events by date
* Highlight past events differently

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## Features

* Responsive Design
* Event Cards Layout
* Form Validation
* Search Functionality
* Dynamic Event Management
